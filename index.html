OnionMail is an open source SMTP/POP3 compatible mail server with some functions designed for Tor hidden services. OnionMail uses filesystem cryptography and some extended functions. This server allows you to use the e-mail in the tor network without losing the ability to e-mail in clearnet.

http://onionmail.info/manual.html

The root directory of project is: /mnt/sdcard/ONM/ Change that setting in etc/onionmail/config.conf

Version 2 is upcoming. This repository is updated automatically from the internal repository development of OnionMail. The this repository is updated in weekly cycles (sometimes). If you want to contribute to the development of the project contact us to have access to trusted internal repository.
